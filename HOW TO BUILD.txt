This folder shows how to setup the ParticleSystem.
To build this project you need to install CMake, and have the Vulkan SDK.


Software: 
https://CMake.org
https://www.lunarg.com/vulkan-sdk/


After installing reboot your machine and run the "cmake -S ./ -B ./build" in the command prompt in the same directory as the cmakelist.
It will build the solution for the ParticleSystem.
