This folder shows how to setup a 3D application purely with the EnTT ECS.
To build this project you need to install CMake, .net Maui, and have the Vulkan SDK.


Software: 
https://CMake.org
https://www.lunarg.com/vulkan-sdk/
https://dotnet.microsoft.com/en-us/apps/maui


After installing reboot your machine and run the "build_all.bat" script.
It will build the solution for the ParticleSystem library and the App
